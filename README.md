# Sol y Cía Ltda - Vite + React + Tailwind

Proyecto web corporativo listo para abrir en Visual Studio Code.

## Requisitos
- Node.js 20.19+ o 22.12+ recomendado por Vite 8.
- npm instalado.

## Instalación
```bash
npm install
```

## Levantar en desarrollo
```bash
npm run dev
```

## Compilar para producción
```bash
npm run build
```

## Vista previa del build
```bash
npm run preview
```

## Personalizaciones recomendadas
- Cambiar el número de WhatsApp en `src/App.jsx`
- Cambiar correo, teléfono y dirección
- Reemplazar testimonios y proyectos de ejemplo
- Reemplazar el logo SVG en `public/logo-sol-y-cia.svg`
